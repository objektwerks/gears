Gears
-----
>Gears feature tests using uJson and Scala 3.

Warning
-------
>**sbt run** fails to link AsyncApp on **java.net.URL**.

Build
-----
1. sbt clean compile

Test
----
1. sbt clean test

Run
---
1. sbt run
